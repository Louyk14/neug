# 报告生成指南

> SKILL.md Step 5 的补充指南。顶层 agent 和 subagent 均可使用。

## 每个子问题的 4 个必备章节

每个子问题 4.X 必须包含以下 4 个章节，缺一不可：

```
4.X [子问题标题]
  4.X.1 查询分析（查询→结论→数据表→图表）
  4.X.2 对照验证（对照设计→对比表→对比图→结论）
  4.X.3 追因分析（提出追因问题→拆解→各子问题分析）
    4.X.3.1 [追因子问题1]
    4.X.3.2 [追因子问题2]
  4.X.4 小结（主分析+对照+追因 → 因果链）
```

## Markdown 标题级别

| 层级 | 格式 | Markdown | 示例 |
|------|------|----------|------|
| 报告大章 | h2 | `##` | ## 4. 详细分析 |
| 子问题 | h3 | `###` | ### 4.1 论文产出变化 |
| 子问题内章节 | h4 | `####` | #### 4.1.1 查询分析 |
| 追因子问题 | h5 | `#####` | ##### 4.1.3.1 为什么产出下降? |
| 第2层追因 | h6 | `######` | ###### 4.1.3.1.3.1 ... |

## 小结的撰写规则（4.X.4）

小结是每个子问题最重要的产出，必须包含：

1. **主分析结论**: 引用 4.X.1 的具体数值
2. **对照结论**: 引用 4.X.2 的差异率
3. **追因结论**: 逐条列出 4.X.3 各追因子问题的结论
4. **因果链**: 将以上串联为因果关系

```markdown
#### 4.1.4 小结

**主分析结论**: 俄罗斯论文产出2021-2023年下降16.8%
**对照结论**: 全球同期下降13.9%，俄罗斯额外下降2.9个百分点
**追因结论**:
- 4.1.3.1: 国际合作论文下降25%是主要原因
- 4.1.3.2: 特定学科(物理)受冲击最大(-42.7%)
**因果链**: 论文产出下降(-16.8%) ← 国际合作大幅减少(-25%) ← 物理等国际合作密集学科受冲击最重
```

## 追因分析的嵌入规则

1. 读取 subagent 保存的分析结果
2. 调整标题级别使其正确嵌套（追因子问题用 h5，第2层用 h6）
3. 确保图片路径统一指向 images/ 目录
4. 将内容插入到 4.X.3.Y 位置

## 最终检查

- [ ] 每个 4.X 都有 4.X.1 + 4.X.2 + 4.X.3 + 4.X.4（四个章节完整）
- [ ] 4.X.3 追因分析非空（有 subagent 分析结果）
- [ ] 4.X.4 包含因果链
- [ ] 标题级别正确递进
- [ ] 执行摘要反映了追因发现
- [ ] 局限性不包含已分析过的内容
